## Welcome to Alerta Ciudadana

Alerta Ciudadana is a comprehensive REST API built with FastAPI that enables citizens to report urban incidents, track their status, and help authorities prioritize emergency responses. The system leverages machine learning to automatically classify incident priorities and detect critical zones based on geographical clustering.

## Technology Stack

The API is built with modern Python technologies:

- **FastAPI** - High-performance async web framework
- **SQLAlchemy** - ORM for database operations
- **Pydantic** - Data validation and serialization
- **scikit-learn** - Machine learning for priority classification and clustering
- **PyJWT** - JSON Web Token implementation
- **Passlib + bcrypt** - Secure password hashing
- **Pandas & NumPy** - Data processing and analysis

Incidents flow through the following states:

```python
class EstadoIncidente(str, Enum):

    descartado = "descartado"         # Dismissed
```

```python
class RolUsuario(str, Enum):
```

## ML-Powered Features

### Priority Classification

The system automatically analyzes incident descriptions and assigns priority levels:

- **High Priority**: Fire, weapons, robbery, accidents, falling objects
- **Medium Priority**: Noise, pollution, garbage, lighting issues
- **Low Priority**: Neighborhood disputes, late-night music

### Critical Zone Detection

Using DBSCAN (Density-Based Spatial Clustering), the API identifies geographical areas with high incident concentration. This helps authorities:

- Allocate resources efficiently
- Identify problem areas requiring intervention
- Track incident trends over time
- Plan preventive measures

<Note>
  The system automatically calculates optimal clustering parameters using nearest neighbor distances and percentile analysis.
</Note>

## Quick Example

Here's how simple it is to report an incident:

```bash
curl -X POST "http://localhost:8000/incidente/" \
  -H "Content-Type: application/json" \
  -d '{
    "titulo": "Bache peligroso en Avenida Principal",
    "descripcion": "Hueco grande que puede causar accidentes",
    "latitud": 19.4326,
    "longitud": -99.1332,
    "estado": "pendiente",
    "usuario_id": 1,
    "categoria_id": 1
  }'
```
