---
title: 'Register User'
api: 'POST /register'
description: 'Register a new user account in the system'
---

This endpoint allows new users to register in the Alerta Ciudadana system. Upon successful registration, a new user account is created with the role of "ciudadano" (citizen) by default.

## Request Body

<ParamField body="nombre" type="string" required>
  User's first name
</ParamField>

<ParamField body="apellidos" type="string" required>
  User's last name(s)
</ParamField>

<ParamField body="correo_electronico" type="string" required>
  User's email address. Must be a valid email format and unique in the system.
</ParamField>

<ParamField body="telefono" type="string" required>
  User's phone number
</ParamField>

<ParamField body="contrasena" type="string" required>
  User's password. Will be hashed before storage.
</ParamField>

## Response

<ResponseField name="usuario" type="object">
  The created user object
  
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Unique identifier for the user
    </ResponseField>
    
    <ResponseField name="nombre" type="string">
      User's first name
    </ResponseField>
    
    <ResponseField name="correo_electronico" type="string">
      User's email address
    </ResponseField>
    
    <ResponseField name="rol" type="string">
      User's role ("ciudadano" or "admin")
    </ResponseField>
    
    <ResponseField name="fecha_creacion" type="string">
      Timestamp when the user was created (ISO 8601 format)
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="message" type="string">
  Success message confirming registration
</ResponseField>

## Examples

<RequestExample>
```bash cURL
curl -X POST https://api.alerta-ciudadana.com/register \
  -H "Content-Type: application/json" \
  -d '{
    "nombre": "Juan",
    "apellidos": "Pérez García",
    "correo_electronico": "juan.perez@example.com",
    "telefono": "+52 55 1234 5678",
    "contrasena": "securePassword123"
  }'
```

```python Python
import requests

url = "https://api.alerta-ciudadana.com/register"
payload = {
    "nombre": "Juan",
    "apellidos": "Pérez García",
    "correo_electronico": "juan.perez@example.com",
    "telefono": "+52 55 1234 5678",
    "contrasena": "securePassword123"
}

response = requests.post(url, json=payload)
print(response.json())
```

```javascript JavaScript
fetch('https://api.alerta-ciudadana.com/register', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    nombre: 'Juan',
    apellidos: 'Pérez García',
    correo_electronico: 'juan.perez@example.com',
    telefono: '+52 55 1234 5678',
    contrasena: 'securePassword123'
  })
})
.then(response => response.json())
.then(data => console.log(data));
```
</RequestExample>

## Error Responses

<ResponseField name="400 Bad Request" type="error">
  Returned when the email address is already registered in the system or when the request body is invalid.
  
  **Error Details:**
  - `Correo ya registrado` - The provided email is already associated with an existing account
</ResponseField>

<ResponseField name="422 Unprocessable Entity" type="error">
  Returned when the request body doesn't match the expected schema (e.g., invalid email format, missing required fields).
</ResponseField>
