## Endpoint

```
POST /comentario/
```

Creates a new comment associated with an incident. Comments allow users to provide updates, feedback, or additional information about incidents.

## Request Body

The request body should contain a JSON object with the following fields:

<ParamField body="contenido" type="string" required>
  The content/text of the comment. Maximum length: 500 characters
</ParamField>

<ParamField body="usuario_id" type="integer" required>
  ID of the user creating the comment. Must reference an existing user in the `usuarios` table
</ParamField>

<ParamField body="incidente_id" type="integer" required>
  ID of the incident this comment belongs to. Must reference an existing incident in the `incidentes` table
</ParamField>

## Response

Returns the created comment object with auto-generated fields.

<ResponseField name="id" type="integer" required>
  Auto-generated unique identifier for the comment
</ResponseField>

<ResponseField name="contenido" type="string" required>
  The content/text of the comment
</ResponseField>

<ResponseField name="usuario_id" type="integer" required>
  ID of the user who created the comment
</ResponseField>

<ResponseField name="incidente_id" type="integer" required>
  ID of the incident this comment belongs to
</ResponseField>

<ResponseField name="fecha" type="datetime" required>
  Auto-generated timestamp when the comment was created (UTC)
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X POST "http://localhost:8000/comentario/" \
  -H "Content-Type: application/json" \
  -d '{
    "contenido": "La situación está mejorando en esta área",
    "usuario_id": 5,
    "incidente_id": 12
  }'
```

```python Python
import requests

data = {
    "contenido": "La situación está mejorando en esta área",
    "usuario_id": 5,
    "incidente_id": 12
}

response = requests.post(
    "http://localhost:8000/comentario/",
    json=data
)
comment = response.json()
```

```javascript JavaScript
const response = await fetch(
  'http://localhost:8000/comentario/',
  {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      contenido: 'La situación está mejorando en esta área',
      usuario_id: 5,
      incidente_id: 12
    })
  }
);
const comment = await response.json();
```
</CodeGroup>

## Relationships

- **usuario_id**: Must reference an existing user in the `usuarios` table
- **incidente_id**: Must reference an existing incident in the `incidentes` table

## Notes

- The `id` field is auto-generated and should not be included in the request
- The `fecha` field is automatically set to the current UTC timestamp
- The `contenido` field has a maximum length of 500 characters
