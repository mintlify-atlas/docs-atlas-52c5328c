## Endpoint

```
GET /comentarios/
```

Retrieves a paginated list of comments associated with incidents. Comments include the content, author, and related incident information.

## Query Parameters

<ParamField query="skip" type="integer" default="0">
  Number of records to skip for pagination
</ParamField>

<ParamField query="limit" type="integer" default="10">
  Maximum number of records to return (pagination limit)
</ParamField>

## Response

Returns an array of comment objects.

<ResponseField name="id" type="integer" required>
  Unique identifier for the comment
</ResponseField>

<ResponseField name="contenido" type="string" required>
  The content/text of the comment (max 500 characters)
</ResponseField>

<ResponseField name="usuario_id" type="integer" required>
  ID of the user who created the comment. References `usuarios.id`
</ResponseField>

<ResponseField name="incidente_id" type="integer" required>
  ID of the incident this comment belongs to. References `incidentes.id`
</ResponseField>

<ResponseField name="fecha" type="datetime" required>
  Timestamp when the comment was created (UTC)
</ResponseField>

## Example Request

<CodeGroup>
```bash cURL
curl -X GET "https://api.example.com/comentarios/?skip=0&limit=10" \
  -H "Content-Type: application/json"
```

```python Python
import requests

response = requests.get(
    "https://api.example.com/comentarios/",
    params={"skip": 0, "limit": 10}
)
comments = response.json()
```

```javascript JavaScript
const response = await fetch(
  'https://api.example.com/comentarios/?skip=0&limit=10',
  {
    method: 'GET',
    headers: {
      'Content-Type': 'application/json'
    }
  }
);
const comments = await response.json();
```
</CodeGroup>

## Relationships

- **usuario_id**: References the `usuarios` table. Each comment is associated with one user (the author).
- **incidente_id**: References the `incidentes` table. Each comment belongs to one incident.

## Pagination

Use the `skip` and `limit` parameters to paginate through results:

- To get the first page: `skip=0&limit=10`
- To get the second page: `skip=10&limit=10`
- To get the third page: `skip=20&limit=10`
