---
title: "Get Category"
api: "GET /categoria/{id}"
---

Retrieves a specific category by its ID.

## Path Parameters

<ParamField path="id" type="integer" required>
  The unique identifier of the category to retrieve
</ParamField>

## Response

Returns a single category object.

<ResponseField name="id" type="integer">
  Unique identifier for the category
</ResponseField>

<ResponseField name="nombre" type="string">
  Category name (e.g., "Robo", "Fuga de agua")
</ResponseField>

<ResponseField name="descripcion" type="string">
  Category description
</ResponseField>

<RequestExample>

```bash cURL
curl -X GET "https://api.example.com/categoria/1" \
  -H "Content-Type: application/json"
```

```python Python
import requests

response = requests.get("https://api.example.com/categoria/1")
print(response.json())
```

```javascript JavaScript
fetch('https://api.example.com/categoria/1')
  .then(response => response.json())
  .then(data => console.log(data));
```

</RequestExample>

<ResponseExample>

```json 200 Success
{
  "id": 1,
  "nombre": "Robo",
  "descripcion": "Incidentes relacionados con robos y hurtos"
}
```

```json 404 Not Found
{
  "detail": "Categoría no encontrada"
}
```

</ResponseExample>
